# AdvGPT

AdvGPT is a polished static landing page concept for an AI decision advisor. It presents a product narrative around turning messy goals, stalled projects, and open questions into ranked recommendations, practical plans, and action-ready briefs.

## What is included

- `index.html` — semantic, accessible landing page markup for the AdvGPT concept.
- `styles.css` — responsive visual system with a dark AI advisor aesthetic.

## Preview locally

Open `index.html` directly in a browser, or serve the folder with Python:

```bash
python3 -m http.server 8000
```

Then visit <http://localhost:8000>.
