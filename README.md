# xvAI

xvAI is a polished static landing page concept for an experimental AI workspace. It presents a product narrative around turning rough ideas into research briefs, structured plans, and launch-ready execution tracks.

## What is included

- `index.html` — semantic, accessible landing page markup for the xvAI concept.
- `styles.css` — responsive visual system with a dark AI workspace aesthetic.

## Preview locally

Open `index.html` directly in a browser, or serve the folder with Python:

```bash
python3 -m http.server 8000
```

Then visit <http://localhost:8000>.
